Production Process

1

Pre-Production Planning

Design meetings, technical planning, equipment assessment, and crew allocation. This phase can take 2-3 months for major productions.

  • Script analysis and technical requirements
  • Lighting and sound design development
  • Set design and construction planning
  • Crew training and skill development
2

Design & Construction

Building sets, programming lighting sequences, setting up sound systems, and creating all technical elements needed for the production.

  • Set construction and painting
  • Lighting rig installation and programming
  • Sound system setup and testing
  • Costume and prop creation
3

Technical Rehearsals

Integration of all technical elements with the performance, fine-tuning timing, and ensuring smooth operation of all systems.

  • Cue-to-cue rehearsals
  • Dress rehearsals with full technical
  • Problem-solving and adjustments
  • Crew coordination and timing
4

Performance Run

Executing the technical elements flawlessly for each performance, maintaining equipment, and adapting to any unexpected challenges.

  • Pre-show equipment checks
  • Live technical operation
  • Quick problem resolution
  • Post-show maintenance
5

Strike & Evaluation

Dismantling sets, storing equipment, conducting post-production evaluations, and planning improvements for future productions.

  • Set dismantling and storage
  • Equipment maintenance and inventory
  • Production evaluation and feedback
  • Skills assessment and development planning
